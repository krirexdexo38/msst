Learn About Complexity

O(1) - Constant Time

Operations that take the same amount of time regardless of input size.

def get_first(arr):
    return arr[0]

O(n) - Linear Time

Time grows linearly with input size.

def linear_search(arr, target):
    for item in arr:
        if item == target:
            return True
    return False

O(log n) - Logarithmic Time

Time grows slowly even for large inputs.

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

O(n²) - Quadratic Time

Time grows quadratically - nested loops.

def bubble_sort(arr):
    for i in range(len(arr)):
        for j in range(len(arr) - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]

O(2^n) - Exponential Time

Time doubles with each additional input element.

def fibonacci_recursive(n):
    if n <= 1:
        return n
    return fibonacci_recursive(n-1) + fibonacci_recursive(n-2)
